Appraisal

Appraisal.

AppraisalDateNonUtc

The appraisal date.
Should only be the date (no time component).
Data type: DateTime
e.g. 2012-03-01

Bathrooms

The number of bathrooms.
Data type: Nullable int
e.g. 2

Bedrooms

The number of bedrooms.
Data type: Nullable int
e.g. 4

Cma

The comparative market analysis.
Data type: Cma

ContactUris

The URIs of the contacts for the appraisal.
Data type: Uri list

CreateDateTime

The creation date time in UTC.
This is updated by the system. Posted values ignored.
Data type: DateTime
e.g. 2012-03-01 10:11:12

CreatedByStaffUri

The uri to the staff member that created the item.
Data type: Uri
e.g. /Staff/4312

DeleteReason

The reason for deletion of this record.
A non null value will mark this item as logically deleted.
Data type: string
e.g. No longer required.

DisplayAddress

The display address.
Data type: string
e.g. 5/123 High Street, Primrose Hill, Sydney, 8001

FloorNumber

The floor number.
Data type: string
e.g. 10

Garden

The name of the garden.
Data type: string
e.g. Belsize Gardens

IsActive

Whether the appraisal is active or not.
This is updated by the system. Posted values ignored.
Data type: Boolean
e.g. true

LandArea

The land area in square metres.
Data type: Nullable decimal
e.g. 765

Latitude

The latitude coordinate of the property.
Data type: Nullable decimal
e.g. -24.60706913770968

ListingTypeUri

The listing type uri.
Data type: Uri
e.g. /ListingType/1

ListingUri

The uri to the listing.
When an appraisal has been converted to a listing this is the uri to the listing.
Data type: Uri
e.g. /Listing/832165

LocationUri

The uri to the location.
Data type: Uri
e.g. /Location/13023

Longitude

The longitude coordinate of the property.
Data type: Nullable decimal
e.g. 133.505859375

MapZoomLevel

The zoom level to display them map.
Data type: Nullable int
e.g. 15

OrganisationalUnitUri

The uri to the organisational unit (office) of the user that created the item.
Data type: Uri
e.g. /OrganisationalUnit/865

PostCode

The post code.
Data type: string
e.g. 8052

PropertyTypeUri

The property type uri.
Data type: Uri
e.g. /PropertyType/17

Recommendations

Recommendations.
Data type: string
e.g. Mow the lawns.

StaffUris

The URIs of the sales consultants for the appraisal.
The first staff uri is the primary sales consultant.
Data type: Uri list

StreetName

The street name.
Data type: string
e.g. High Street

StreetNumber

The street number.
Data type: string
e.g. 43

Summary

The summary
Data type: string

UnitNumber

The street address unit number.
Data type: string
e.g. 3

UpdateDateTime

The date and time in UTC that this item was updated.
This is updated by the system. Posted values ignored.
Data type: Date time
e.g. 2012-03-01 10:11:12

UpdatedByStaffUri

The Uri of the staff member that updated the item.
This is updated by the system. Posted values ignored.
Data type: Uri
e.g. /Staff/53326
